# 校园二手交易平台 Starter

这是《Web 程序设计》课程大作业的基础框架，已包含：

- Spring Boot + Spring MVC + MyBatis + Thymeleaf
- PostgreSQL 数据库脚本
- 登录 / 注册 / 退出
- 普通用户发布商品、上传商品图片
- 商品列表、关键词/分类筛选、分页
- 商品详情
- 管理员审核商品
- 订单管理：买家下单、卖家确认、买家完成、双方取消
- 用户个人订单页：我买到的订单、我卖出的订单
- 管理员订单管理页和最近订单列表
- ECharts 商品分类饼图、最近 7 天订单柱状图

## 环境

- JDK 8 或更高
- Maven 3.6+
- PostgreSQL 12+

## 启动步骤

1. 创建数据库：

```sql
CREATE DATABASE secondhand_db;
```

2. 连接 `secondhand_db`，执行：

```sql
sql/schema.sql
```

3. 修改 `src/main/resources/application.yml` 中的数据库用户名和密码。

4. 启动项目：

```bash
mvn spring-boot:run
```

5. 浏览器访问：

```text
http://localhost:8080
```

## 订单流程

1. 买家在商品详情页点击“我要购买”，系统生成 `WAIT_CONFIRM` 订单，并把商品状态改为 `SOLD`，防止重复购买。
2. 卖家在“我的订单”中点击“确认交易”，订单进入 `TRADING`。
3. 买家线下面交后点击“确认完成”，订单进入 `FINISHED`。
4. 未完成前，买家或卖家都可以取消订单，取消后商品恢复为 `ON_SALE`。
